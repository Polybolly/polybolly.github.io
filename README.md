## Run Locally

Clone the project

```bash
  git clone https://github.com/Polybolly/polybolly.github.io
```

Go to the project directory

```bash
  cd Polybolly
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
